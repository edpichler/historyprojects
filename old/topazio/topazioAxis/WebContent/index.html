JBuilder Apache Axis Admin Console

Welcome to the JBuilder Apache Axis Admin Console based on Axis 1.2.1

First, configure the project for web services using the Axis Web Services Server Configuration wizard (File|New|Web Services). Build and run the project with the Web Services Server runtime configuration to automatically deploy the archives to the Axis server. After you've deployed a web service, you can use the Axis Admin Console to execute a variety of tasks.

Choose one of the following links:

  • View the list of deployed web services
  • Validate the local installation's configuration
    [If this doesn't work, see Validating Axis.]
  • Administer Axis
    [Disabled by default for security reasons. See Enabling Axis administration.]
  • SOAPMonitor
    [Disabled by default for security reasons. See Enabling SOAP monitoring features.]
  • Visit the Apache Axis Home Page
  • Visit the Apache Wiki

For more information about how to use the Axis toolkit in JBuilder, see Developing Web Services in the JBuilder documentation.
